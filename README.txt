En este proyecto buscamos darle soporte a una tienda de computadoras ficticia. Para esto implementamos una BD en SQLSERVER con varias funcionalidades como store procedures, triggers y demás. Por otro lado realizamos una WebAPI para consumir en un frontend desarrollado con HTML,CSS,JS y BOOTSTAP.
Dentro de las funcionalidades más destacadas de la aplicación podemos encontrar, un panel de DASHBOARD que muestra información relevante al usuario, la gestión para la carga de pedidos con varios detalles, un listado con filtro para poder visualizarlos, también brindamos soporte para la creación, modificación, y eliminación de componentes y un inicio de sesión con credenciales.
